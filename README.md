# 🎮 Hangman Game (Python)

## 📌 Description

This is a simple Python-based Hangman game where players guess a randomly selected word one letter at a time with limited attempts.

## 🚀 Features

* Random word selection
* Letter-by-letter guessing
* Limited incorrect attempts
* Win/Loss conditions
* Simple command-line interface

## 🛠️ Technologies Used

* Python 3

## 📂 Project Structure

```
hangman-game/
│
├── hangman.py
└── README.md
```

## 🎯 Game Rules

* Guess one letter at a time
* You have limited attempts
* Correct guesses reveal the word
* Too many wrong guesses → Game Over
